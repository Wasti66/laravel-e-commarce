# 🛒 E-commerce Laravel Project

A fully functional e-commerce web application built using Laravel.

## 🔥 Features
- User Authentication
- Add to Cart System
- Order & Checkout
- Payment Integration

  ## 🛠️ Technology
- Laravel

## ⚙️ Installation
```bash
git clone https://github.com/Wasti66/laravel-e-commarce.git
cd laravel-e-commarce
composer install
npm install && npm run dev
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
